== Cloud Strategy

[arabic]
. **Workload placement** — right cloud for right workload
. **Identity federation** — single identity across clouds
. **Network connectivity** — secure, low-latency connections
. **Cost optimization** — reserved instances, spot, autoscaling
. **Security posture** — consistent policies across clouds
